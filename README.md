## Project Smart Meter UI🚀

In this code is web application for user interface smart meter with machine learning, computer vision and internet of thing. In this UI, i am using Laravel and Tailwind CSS

## Description UI🚀

1. A prediction system on the use of electrical energy using the LSTM algorithm models based on the website
2. Input token in smart meter with robotic arms IoT
3. Monitoring usage electricity
